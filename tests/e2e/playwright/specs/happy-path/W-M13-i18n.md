## Architecture

- **Library**: react-i18next
- **localStorage key**: `quizLanguage` (từ `utils/quizLanguage.ts`)
- **Toggle**: `QuizLanguageToggle` trong `AppLayout`
- **Supported**: `vi` (default), `en`

## W-M13-L2-001 — Default language vi: page loads với Vietnamese strings

**Priority**: P0
**Est. runtime**: ~3s
**Auth**: storageState=tier3
**Tags**: @happy-path @i18n @parallel-safe

**Setup**:
- Clear localStorage: `page.evaluate(() => localStorage.clear())`

**Actions**:
1. `page.goto('/')`

**Assertions**:
- `localStorage.getItem('quizLanguage')` === null OR 'vi'
- Welcome text in Vietnamese (e.g. "Chào mừng", "Luyện tập", "Thi đấu")

## W-M13-L2-003 — Language persists across page reload

**Priority**: P0
**Est. runtime**: ~4s
**Auth**: storageState=tier3
**Tags**: @happy-path @i18n @write @serial

**Setup**:
- Set `localStorage.quizLanguage = 'en'` trước khi goto

**Actions**:
1. `page.goto('/')`
2. `page.reload()`

**Assertions**:
- UI still English after reload
- `localStorage.getItem('quizLanguage')` === 'en' (still)

**Cleanup**:
- Reset localStorage to 'vi'

## W-M13-L2-005 — API response language param: GET /api/daily-challenge?language=en → English questions

**Priority**: P1
**Est. runtime**: ~4s
**Auth**: no auth (guest)
**Tags**: @happy-path @i18n @parallel-safe

**Actions**:
1. `GET /api/daily-challenge?language=en`

**API Verification**:

- But `content` field different:

  - `vi`: Vietnamese question text
- Verify at least 1 question có distinct translation
