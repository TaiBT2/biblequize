**Cleanup**: none

**Notes**:

### W-M13-L1-002 — Switch language EN → UI text thay đổi

**Priority**: P1
**Est. runtime**: ~4s
**Auth**: storageState=tier3
**Tags**: @smoke @i18n

**Setup**:
- Set ngôn ngữ mặc định về VI: `localStorage.setItem('quizLanguage', 'vi')`

**Preconditions**:
- Ngôn ngữ hiện tại là VI

**Actions**:
1. `page.goto('/')`
2. `page.waitForSelector('[data-testid="lang-toggle-en"]')`
3. `page.getByTestId('lang-toggle-en').click()`

**Assertions**:
- `expect(page.getByTestId('lang-toggle-en')).toHaveAttribute('data-active', 'true')` ← EN active
- `expect(page.getByTestId('home-greeting')).toHaveText(/Good (morning|afternoon|evening)/i)` ← EN text

**Cleanup**:
- `page.evaluate(() => localStorage.setItem('quizLanguage', 'vi'))` ← restore VI

**Notes**:
- `i18n.changeLanguage('en')` được gọi + `localStorage['quizLanguage'] = 'en'`
- [NEEDS TESTID: lang-toggle-en] với `data-active` attribute

### W-M13-L1-003 — Language preference persist qua reload

**Priority**: P1
**Est. runtime**: ~5s
**Auth**: storageState=tier3
**Tags**: @smoke @i18n

**Setup**: none

**Preconditions**:
- User đã đăng nhập

**Actions**:
1. `page.goto('/')`
2. `page.getByTestId('lang-toggle-en').click()` ← switch sang EN
3. `page.reload()`
4. `page.waitForSelector('[data-testid="home-greeting"]')`

**Assertions**:
- `expect(page.getByTestId('home-greeting')).toHaveText(/Good (morning|afternoon|evening)/i)` ← EN vẫn active sau reload
- `expect(await page.evaluate(() => localStorage.getItem('quizLanguage'))).toBe('en')`

**Cleanup**:
- `page.evaluate(() => localStorage.setItem('quizLanguage', 'vi'))`

**Notes**:
- Language detection dựa vào `localStorage['quizLanguage']` (storage key từ i18n config)
